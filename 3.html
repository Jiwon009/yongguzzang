JavaScript , [6] often abbreviated as JS, is high-level, interpreted programming language. it is a language which is also characterized as dynamic, weakly typed, prototype-based and multi-paradigm. Alongside HTML and CSS, Javascript is one of the three core technologies of the World Wide Web. [7] JavaScript enables interactive web pages and thus is an essential part o web applications. the vast majority of websites use it, [8] and all major web browers have a dedicated Javascript engine to execute it.
